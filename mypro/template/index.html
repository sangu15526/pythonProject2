Student Engagement Recognition System for an E-learning Environment

About

This recognition system was developed as a part of our S8 Project (CS 492) under the guidance of our mentor.


Problem Statement

Google Meet and Zoom has helped us to attend classes virtually. But conducting online classes is not as efficient as offline classes. In platforms like Google meet, the teacher has to swipe back and forth between screens while taking class to check if all students are attentive. Teachers need to have more than a pair of eyes to teach and verify whether each of the students are attentively listening. If a student is not attentive in class, the existing systems does not automatically send alert to the teacher. Therefore, there is no motivation for students to concentrate on class as there is no one monitoring them. So we have devised a system which to detect the engagement of the student using CNN. An alert system and a student engagement calculator has been incorporated with the algorithm so as to send alert and warning to teachers and students respectively if students are not engaged in the class.

Facial Posture Detection

The first step was to extract facial features of the user. First, we started by learning about the CAT vs DOG classifier. We referred a to code that created an Image Classifier which can distinguish whether a given pic is of a dog or cat or something else depending upon the fed data. Then we used the base code of a emotion detection model, that can identify emotions such as “angry”, “disgust”, “fear”, “happy”, “sad”, “surprise”, “neutral”. We then found methods to incorporate this feature in to our facial posture detector to detect the emotions of the students.

Alert System

The alert system acts as the base for understanding the engagement of the student. The students engagement is stored at regular intervals (eg: 5 mins, 10 mins, 15 mins) depending on the state the student (engaged, bored, confused, frustrated) while the data is being recorded into the database during online class. The tutor can access the details through the admin login.

Student Engagement Calculator

Once the class is over, the tutor can view the statistics of the class. For example, having 80% or more students engaged means the overrall class was engaging, while 60% to 80% implying confusion, and similarly reducing for the various engagements shown by the calculator. The tutor can thus take effective measures to improve the quality of online classes and give more attention to students who are less engaged.

Objective

The objective of this project is to study students online course engagement recognition from students appearance and behavioral information using deep learning methods. Both spatio and temporal features of a student are considered for evaluation using Convolutional Neural Networks on the DAiSEE Dataset. The proposed system judges students engagement levels by checking if the student is looking somewhere else or has dozed off in between. It checks if the user is both physically and mentally present in class. This system increases the efficiency of online classes.
